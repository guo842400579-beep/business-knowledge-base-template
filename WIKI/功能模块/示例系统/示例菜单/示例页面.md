# 示例页面

> 本页描述虚构业务，不代表任何真实系统或业务规则。

示例页面展示“示例对象”的筛选区、列表和按状态出现的操作入口。

## 页面定位

| 项目 | 内容 |
|------|------|
| 所属系统 | 示例系统 |
| 菜单路径 | 示例系统 / 示例菜单 / 示例页面 |
| 主要角色 | 未由截图确认 |
| 页面用途 | 查询和查看示例对象列表；其他入口行为待验证 |
| 关联流程 | 示例流程 |

## 筛选条件

| 可见标签 | 控件类型 | 截图证据 |
|------|----------|------|
| Keyword | 输入框 | 当前显示 `DEMO-001` |
| Status | 下拉框 | 当前显示 `All`；可选值未展开 |
| Created Date | 日期输入 | 当前显示 `2026-07-02` |
| Search | 按钮 | 位于筛选区 |
| Reset | 按钮 | 位于筛选区 |

## 列表字段

| 可见列头 | 对应含义 | 备注 |
|------|------|------|
| Select | 行选择 | 当前三行均未勾选 |
| Demo ID | 示例编号 | 显示 `DEMO-001` 等虚构值 |
| Demo Name | 示例名称 | 显示虚构对象名称 |
| Status | 示例状态 | 截图样本含 `Pending`、`Completed`、`Draft` |
| Updated At | 更新时间 | 截图样本精确到分钟 |
| Actions | 行操作 | 不同状态行显示的入口不同 |

## 字段和概念关系

| 页面字段 | 对应概念 | 说明 |
|----------|----------|------|
| Demo ID | 示例编号 | 与字段模板中的“示例编号”对应 |
| Demo Name | 示例名称 | 与字段模板中的“示例名称”对应 |
| Status | 示例状态 | 页面英文值与中文业务状态尚未完整映射 |

## 可见操作入口

| 可见标签 | 位置 | 截图状态 | 已确认边界 |
|----------|------|----------|------------|
| New | 列表顶部 | 当前截图可见 | 仅确认入口存在，打开形态和提交结果未实采 |
| Export | 列表顶部 | 当前截图可见 | 仅确认入口存在，导出字段和数据范围未实采 |
| View | 行操作 | 三种状态均可见 | 与专家补充中“待处理可查看”一致 |
| Edit | 行操作 | `Completed` 样本可见 | 不能据此推断所有已完成对象均可编辑 |
| Delete | 行操作 | `Draft` 样本可见 | 仅确认入口存在，删除方式和确认层级未实采 |

## 页面在流程中的作用

| 流程步骤 | 页面证据 | 结论 |
|----------|----------|------|
| 定位对象 | Keyword、Status、Created Date、Search | 页面具备筛选入口，具体匹配规则未实采 |
| 查看状态 | 列表 Status 列 | 可见三个英文状态样本，不代表枚举全集 |
| 查看对象 | 三行均有 View | 页面提供查看入口 |
| 判断下一步 | 操作入口随状态样本不同 | 状态与操作存在关联，但完整规则待确认 |

## 信息边界

- 截图没有展示下拉选项、弹窗、侧拉框、确认文案、导出文件或操作结果。
- `Pending`、`Completed`、`Draft` 只是当前列表样本，不能当作完整枚举。
- `Draft` 与中文“待处理”的对应关系尚未确认，见[待确认项](../../../待确认项/待确认项_index.md)。
