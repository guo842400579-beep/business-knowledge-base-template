# 术语对照表

本表给出用户口语、旧名称和系统正式名称之间的映射。

| 页面英文 | 中文业务称呼 | 说明 | 相关文档 |
|----------|--------------|------|----------|
| Demo Object | 示例对象 | 页面列表对象与专家原话中的对象称呼 | [示例概念](./示例概念.md) |
| Demo ID | 示例编号 | 字段模板中的文本编号 | [示例概念](./示例概念.md) |
| Demo Name | 示例名称 | 字段模板中的对象名称 | [示例概念](./示例概念.md) |
| Status | 示例状态 | 页面值与中文枚举尚未完整对应 | [示例概念](./示例概念.md) |

## 解释边界

- `Completed` 与“已完成”在字面和现有样本上相符，但完整枚举映射仍需确认。
- `Pending`、`Draft` 不能在缺少业务口径时直接合并为“待处理”。
- 本表中的词汇均为虚构样本，不代表任何真实系统词汇。
