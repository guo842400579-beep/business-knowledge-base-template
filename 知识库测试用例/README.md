# 知识库查询回归用例

本目录保存完全虚构的查询问题和期望答案，用于验证知识库结构或查询 Skill 变更后，回答仍遵守来源、证据和待确认边界。

## 目录

```text
知识库测试用例/
├── 问题/
└── 答案/
```

问题文件只描述用户问法；答案文件记录必须覆盖的结论、来源和禁止推断项。不要把真实客户、人员、系统或生产问题放入可分享模板。

## 使用方式

1. 让一个未看到答案文件的智能体按 `skills/query/SKILL.md` 回答问题。
2. 对照答案检查：
   - 是否先从 WIKI 正文回答；
   - 是否引用正确 Index 和具体业务页；
   - 是否把待确认项当成限制而非事实；
   - 是否避免默认读取 `.work`、RAW、log 和备份；
   - 是否明确知识缺口，没有编造。
3. Skill、索引、术语或证据边界发生大改后重跑。

回归用例不是业务事实来源，不应被日常业务查询引用。
